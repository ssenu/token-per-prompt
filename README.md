# token-per-prompt

**질문(턴)별 토큰 사용량**을 보여주는 Claude Code 플러그인. 세션 총량이 아니라 *이 질문 하나가 얼마나 썼는지*를 답변 아래·상태줄·리포트 표로 보여줍니다.

- 🧮 **답변 아래(Stop 훅)** — 매 답변 끝에 그 질문의 토큰을 한 줄로 기록
- 📊 **상태줄(실시간)** — 하단에 현재 턴 값 실시간 표시(🔄 진행중 / ✅ 완료), 기존 상태줄 위에 유지
- 📋 **리포트 표** — 세션 전체 질문을 한 표로 (`/token-report`)
- 🔒 **100% 로컬 · 토큰 소모 0** — 로컬 transcript만 읽음
- 🖥️ **크로스플랫폼** — macOS / Windows, 외부 의존성 없음(파이썬 3.8+ 표준 라이브러리)

## 표시 예시

```
답변 아래 :  출력 1,234 · 새 컨텍스트 80 · 캐시읽기 320,000 | 5시간 리밋의 약 0.25%
상태줄    :  ✅ 출력 1,234 · 새 컨텍스트 80 · 캐시읽기 320,000 | 5시간 리밋의 약 0.25%
```

리포트(`/token-report`):

```
시각           출력       새컨텍스트     5시간%   질문
────────────────────────────────────────────────────────────────────
14:06     5,165      12,335    2.92%   질문 별로 토큰 사용량 보여줘
13:48     2,823     103,409   17.71%   노트북lm 사용법 알려줘
────────────────────────────────────────────────────────────────────
합계        7,988     115,744   20.63%   (2개 질문)
```

## 설치

```bash
# 1) 마켓플레이스 추가
claude plugin marketplace add ssenu/token-per-prompt
# 2) 플러그인 설치 (전역)
claude plugin install token-per-prompt@token-per-prompt -s user
```

설치 후 Claude Code 안에서:

```
/token-setup      # 표시 위치·수준·요금제·테마 등 초기설정
/token-report     # 현재 세션 질문별 표
```

또는 자연어로 "토큰 사용량 표 보여줘", "토큰 스킬 설정해줘" 라고 해도 됩니다.

## 설정 항목 (`/token-setup`)

| 항목 | 값 |
|------|-----|
| 표시 위치 | `hook`(답변 아래) / `statusline`(상태줄) / `both` |
| 표시 수준 | `simple`(총토큰) / `basic`(종류별) / `detailed`(+5시간%) |
| 요금제 | `pro` / `max5` / `max20` |
| 테마 | `tokyonight` / `default`(무색) |
| 5시간 % | `on` / `off` |
| 자동보정 | `on`(서버 실측으로 리밋 보정) / `off` |
| 새로고침 | 상태줄 갱신 주기(초) |

설정은 `~/.claude/token-per-prompt/config.json`에, 배선은 `~/.claude/settings.json`에 저장됩니다.

## 동작 원리

- **토큰 집계**: 세션 transcript(`~/.claude/projects/**/**.jsonl`)에서 한 질문(마지막 사용자 메시지 이후의 모든 assistant 단계)의 usage를 합산.
- **5시간 %**: `(출력 + 새 컨텍스트 + 입력) ÷ 5시간 리밋`. 캐시읽기는 정가의 ~10%라 제외.
  요금제별 추정 리밋을 기본값으로 쓰고, **자동보정**을 켜면 Anthropic `oauth/usage`의 실제 utilization이 1% 오를 때마다 그 사이 토큰으로 리밋을 역산해 보정(서버 호출은 5분 캐시).
- **상태줄 ✅/🔄**: Stop 훅이 완료 마커를 남기고, 상태줄이 현재 턴이 그 마커와 같은지로 판단.

## 값에 대한 참고

- **출력 + 새 컨텍스트 = 실질 소비**, **캐시읽기 = 정가의 ~10%(싸지만 0은 아님)**.
- **상태줄 값 ≠ 답변 아래 값**일 수 있음 — 상태줄은 "실시간 최신 턴", 답변 아래는 "그 질문의 고정 기록". 상태줄이 ✅(완료)이고 같은 턴이면 일치.
- 5시간 %는 **추정**입니다(Anthropic 리밋은 비공개·동적). 자동보정으로 근사치를 맞춥니다.
